# Mac Text Actions 需求文档

## 1. 项目背景
macOS 用户在日常工作中经常需要处理 JSON、时间戳、日期字符串和普通文本摘要。现有方案通常依赖网页工具、命令行或多个独立应用，操作链路长，并且会打断当前上下文。

`Mac Text Actions` 的目标是在系统层提供一个轻量、稳定、可预测的文本处理入口。用户在任意应用中选中文本后，通过 `global shortcut` 即可快速得到处理结果。

## 2. 产品目标
- 缩短常见文本处理动作路径
- 保持“选中即处理”的系统级体验
- 对低歧义内容给出即时 `primary result`
- 对工具型能力保留明确的 `secondary action`
- 在失败场景下提供可解释反馈

## 3. 目标用户
- 开发者
- 测试工程师
- 产品经理
- 经常处理 JSON、时间、哈希和提醒事项的 macOS 用户

## 4. 典型场景
### 4.1 JSON 格式化
用户在任意应用中选中一段 JSON，按下 `global shortcut` 后，`result panel` 直接展示格式化后的内容。

### 4.2 时间戳转日期
用户选中 `10` 位或 `13` 位时间戳，按下 `global shortcut` 后，`result panel` 直接展示本地日期时间。

### 4.3 日期转时间戳
用户选中可解析的日期字符串，按下 `global shortcut` 后，`result panel` 直接展示对应时间戳。

### 4.4 普通文本计算 MD5
用户选中普通文本，按下 `global shortcut` 后，`result panel` 展示可执行工具，其中 `MD5` 为 `secondary action`。

### 4.5 快速创建提醒事项
用户选中文本后，通过 `result panel` 中的 `Create Reminder` 动作快速创建提醒事项，并使用原文预填标题。

## 5. 功能范围
### 5.1 v1 范围
- 全局快捷键注册与触发
- 当前 `selected text` 读取
- 内容类型识别
- `JSON` 格式化
- `JSON` 合法性校验
- 时间戳转本地日期时间
- 日期字符串转时间戳
- 文本 `MD5` 计算
- 快速创建提醒事项
- 结果复制
- 结果替换原文

### 5.2 非目标
- 剪贴板管理
- 自动替换原文
- 批量处理
- 富文本处理
- 多动作编排
- 提醒事项自然语言时间解析

## 6. 核心规则
- `global shortcut` 优先处理当前 `selected text`
- 当 `selected text` 无法读取时，允许自动回退到剪贴板文本
- 发生 `clipboard fallback` 时，`result panel` 必须明确标注当前结果来源
- 自动识别只决定 `primary result` 的渲染
- 用户未明确操作前，不自动写回原文
- `JSON` 默认只做格式化
- `JSON Compress` 为 `secondary action`
- `MD5` 为 `secondary action`

## 7. 成功标准
- 用户在一次快捷键触发后即可看到结果或错误反馈
- `JSON`、时间戳、日期字符串在高频场景下能稳定识别
- 失败场景无静默无响应
- 主路径足够短，不强迫用户切换上下文

## 8. 约束与依赖
- 依赖 macOS 对全局快捷键和相关系统能力的支持
- 某些应用可能无法稳定提供 `selected text`
- 剪贴板回退需要避免让用户误以为当前结果来自实时选区
- 提醒事项能力依赖系统授权

## 9. 后续扩展方向
- 更多文本处理类型
- 用户自定义快捷键和默认动作
- 更丰富的结果渲染
