# Mac Text Actions 设计文档

## 1. 概述

Mac Text Actions 是一个类似 Raycast/Alfred 的 macOS 文本工具，通过**选中文字 + 按空格键**触发，支持时间戳转换、JSON格式化、MD5、URL编解码等功能。

## 2. UI 风格

- **液态玻璃 (Liquid Glass)** - Swift 6 新特性
- 使用 `.glass` modifier 实现毛玻璃效果
- 跟随系统配色（浅色/深色自动适配）
- SF Pro 系统字体

## 3. 交互流程

### 3.1 快捷键触发模式

**触发条件**：
1. 用户选中文本（通过 Accessibility API 检测）
2. 按一次空格键

**触发后**：
- 在光标位置显示浮动气泡（Popover）
- 气泡内显示检测结果和转换操作按钮
- 支持：复制结果 / 替换选中文字 / 关闭

### 3.2 主窗口模式

**打开方式**：点击菜单栏图标

**布局**：左侧边栏 + 内容区（类似 Settings.app）
- 边栏：功能列表图标导航
- 内容区：对应功能的设置和操作界面

## 4. 功能模块

| 功能 | 触发方式 | 说明 |
|------|----------|------|
| 时间戳/日期转换 | 快捷键气泡 | 检测10/13位时间戳或日期字符串，自动识别并转换 |
| JSON 格式化 | 主窗口 | 输入JSON → 格式化输出 |
| MD5 加密 | 主窗口 | 输入文本 → MD5哈希 |
| URL 编解码 | 主窗口 | URL decode/encode 互转 |

## 5. 技术架构

```
┌─────────────────────────────────────────┐
│ App Shell                               │
│ - 菜单栏图标 (NSStatusItem)              │
│ - 设置窗口 (NSWindow + SwiftUI)          │
├─────────────────────────────────────────┤
│ InputBridge (Accessibility API)         │
│ - 读取当前选中文本                        │
│ - 监听键盘事件 (空格键)                   │
├─────────────────────────────────────────┤
│ PopoverController                        │
│ - 浮动气泡窗口                           │
│ - 液态玻璃样式                           │
├─────────────────────────────────────────┤
│ MacTextActionsCore                      │
│ - ContentDetector (内容检测)             │
│ - TransformEngine (转换引擎)             │
│   - TimestampTransform                   │
│   - JsonTransform                       │
│   - Md5Transform                        │
│   - UrlTransform                        │
└─────────────────────────────────────────┘
```

## 6. 检测优先级

1. 10/13 位数字 → 时间戳
2. 可解析日期字符串 → 格式化日期
3. 有效 JSON → 提示JSON格式化
4. 普通文本 → 提示MD5/URL操作

## 7. 系统权限

- **辅助功能权限**：读取选中文本、监听键盘事件
- **提醒权限**（未来扩展用）

## 8. 界面设计

### 8.1 气泡样式 (Popover)

- 圆角矩形，液态玻璃效果
- 最大宽度 320pt
- 显示：输入内容 → 检测类型 → 转换结果
- 操作按钮：复制 | 替换 | 关闭

### 8.2 主窗口样式

- 左侧边栏宽度 180pt
- SF Symbols 图标 + 文字标签
- 内容区使用液态玻璃背景
- 支持系统深色/浅色模式

## 9. 状态管理

- 当前选中文本
- 气泡可见性
- 活动功能模块
- 转换结果缓存
